# Restaurant Data Analytics

This project analyzes restaurant sales data using Python and Pandas to extract business insights.

## Key Objectives
- Calculate revenue by restaurant
- Analyze order quantity trends
- Identify top performing restaurants
- Study rating patterns
- Perform grouped analysis using Pandas
- Create visualizations using Matplotlib

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google colab

## Key Insights
- Dominos generated the highest revenue.
- Veg category dominated orders.
- Certain restaurants showed better rating consistency.
- Revenue comparison helps identify business performance gaps.
